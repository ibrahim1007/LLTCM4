Panduan Singkat & Cara Membaca Data

Cara Penggunaan

1
Upload Data: Masukkan file Excel/CSV riwayat trading Anda.
2
Isi Modal & Risiko: Tentukan modal awal dan berapa persen risiko per trade.
3
Target Bulanan (%):
- Isi Angka (misal 5): Mode Smart. Compounding lot HANYA JIKA target tembus.
- Kosongkan: Mode Flat. Kalkulasi biasa. Tidak ada compounding sama sekali. Lot risiko selalu berdasarkan modal awal.

Arti Warna di Tabel (Mode Target)

  • Hijau (Target Tercapai)

    Saldo Naik + Lot Bulan Depan NAIK.

  • Oranye (Profit Tapi < Target)

    Saldo Naik (Uang masuk) + Lot Bulan Depan TETAP.

  • Merah (Rugi)

    Saldo Turun. Dalam mode Flat, Lot tetap.

Bedah Logika Teknis

Sistem memisahkan logika berdasarkan Input Target:

1. Mode Flat (Target Kosong)

Kalkulasi keuntungan biasa (Simple Calculation). Basis Resiko (Lot) SELALU SAMA dengan Modal Awal. Tidak ada compounding.

Risk Base = Start Capital (Fixed)

2. Mode Capped (Target Diisi)

Basis Resiko hanya naik jika Profit >= Target. Jika Oranye, saldo naik tapi resiko lot bulan depan tidak naik.

Risk Update if >= Target

3. Pertumbuhan ROI

Persentase pertumbuhan (tabel %) dihitung akumulatif berdasarkan Modal Awal (Input). Bukan dari saldo berjalan.

ROI = Sum(PnL) / Input Capital * 100
